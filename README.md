# Lumo — Intelligent Academic Operating System

Lumo is a customizable academic platform that helps educational institutions manage learning, collaboration, and administration in one unified ecosystem.

Designed for colleges and universities, Lumo brings together students, faculty, HODs, administrators, and AI-powered tools through a centralized platform that can be tailored to each institution's needs.

## Key Features

### For Students

* AI-powered academic assistant
* Collaborative study rooms
* Shared notes and flashcards
* Quiz generation and practice
* Placement preparation resources

### For Faculty

* Resource and assignment management
* Student progress monitoring
* Assessment and quiz workflows
* Department-wide communication

### For HODs & Administrators

* Role-based access control
* Academic analytics and insights
* Faculty and student management
* Institution-wide configuration and moderation

## Technology Stack

```text
Frontend   : React + TypeScript
Backend    : Node.js + Express
Database   : MongoDB
Real-Time  : Socket.io
AI         : Google Gemini API
Auth       : JWT + RBAC
```

## Vision

Lumo aims to become a complete academic operating system that institutions can customize and deploy across departments, campuses, and learning environments.
